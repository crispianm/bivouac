# bivouac: wild campsite locator
This is a tool to rank the viability of wild camping locations, and also calculate off-path routes to them.

## Usage
Run `docker build --tag bivouac .`, and open http://127.0.0.1:5000/

## Requirements
- numpy
- matplotlib
- pandas
- json
- OSGridConverter
- scipy
- scikit image
- shapely
- tqdm
- flask
- mercantile
- overpy
- requests
- ast
- folium
- seaborn
- plotly
